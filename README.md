<h1>
  Alpagotchi
</h1>
<p>
  Alpagotchi (Alpaca + Tamagotchi) allows any user to create and take care of an individual alpaca. 
  Items can be bought to satisfy hunger and thirst and work can be done to earn money for the items.
</p>
